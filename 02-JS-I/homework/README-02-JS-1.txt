Variables: Lugar almacenado en la memoria de la máquina. Tiene un nombre, elegido por el programador, y en ese lugar se puede almacenar un dato para luego utilizarlo para distintas operaciones.
Por ejemplo: se define la variable num, y se almacena dentro de esa variable el dato numérico 2. Esa variable luego puede ser sumada, multiplicada, elevada a una potencia, etc. 
Como su nombre lo indica, esa variable puede ser modificada, pues es "variable". Por lo tanto, puedo hacer ahora que la variable num sea 3 en vez de 2, definiendo su nuevo valor.

Strings: Son datos del tipo cadena de caractéres. Al ser asignados a la variables se los colocan entre comillas. Ejemplos de strings: "Hola" , 'HOLA, COMO ESTAS', "Mi nombre es Federico Roque", '11'.

Funciones (argumentos, `return`): Las funciones son porcions de código que sirven para ser reutilizados cada vez que se los invoca. Muy necesario para operaciones que se repiten en varias partes del programa, por lo tanto no debemos repetir lineas de código. Además, si en algún momento se quiere modificar dicha función, solo se lo haría una vez (y no cada vez que se invoca).
Los argumnetos de la funcion son los valores de entrada que se le dan a una función para poder inicializarla. Es decir, los valores que necesita una función para poder ejecutarse correctamente (Vale aclararar que no tadas las funciones tienen argumentos de entrada.)
El "return" es el valor de salida que otorgará la función. Además, con esta expresión se la finaliza. Todo codigo que esté escrito dentro del scope de la función que esté después del "return" no será ejecutado.

Declaraciones `if`: El if es una expresion condicional. Es decir, si cierta condición se cumple (es verdadera) se ejecuta una porción de código. Por el contrario, si la condición no se cumple (es falsa) no se ejecuta la porción de codigo.

Valores Booleanos: Los valores booleanos son un tipo de dato que tienen dos valores posible. Es el tipo de dato lógico que puede arrojar solo dos valores: true y false (verdadero y falso)
